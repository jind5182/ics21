/* 


 * VM作业：编写虚页页号遍历程序，该程序打印出本进程所能够访问的所有虚页页号
 
 
 * /proc/pid/maps
 
 
 */
 

#include <stdio.h>


#include <sys/types.h>


#include <unistd.h>


int main()


{


	unsigned long start, end;
	
	
	char c; 
	
	
	char s[500];
	
	
	char filename[50];
	
	
	FILE *pf = NULL;
	
	
	
	pid_t pid = getpid();
	
	
	sprintf(filename, "/proc/%ld/maps", pid);
	
	
	pf = fopen(filename, "r");
	
	
	
	while(fscanf(pf, "%[^\n]", s) > 0)
	
	
	{
	
	
		sscanf(s, "%lx-%lx %c", &start, &end, &c);	// 读区间起止和权限
		
		
		start = start >> 12;	// 页号
		
		
		end = end >> 12;
		
		
		if(c == 'r')
		
		
		{
		
		
			for(;start < end; start++)
			
			
				printf("%013lx\n", start);
				
				
		}
		
		
		fscanf(pf, "%c", &c);	// 结尾\n
		
		
	}
	
	
	
	fclose(pf);
	
	
	return 0;
	
	
}
